This program can be run using Jupyter locally in any modern web browser. To run 
it, simply start a jupyter environment within the root project directory and 
run Allen_Project1, which is the notebook containing all of the project code.
Once the notebook is started, simply execute all cells within the notebook.
Make sure that the data files are available from the root project directory 
within the 'Mushroom' folder. 

Dependencies: python3, pandas, numpy, sklearn
